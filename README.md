# YelpCamp
Live Deploy at: https://w-d-b-yelp-camp.herokuapp.com/
<br>
# Description
A Yelp-like web app. <br>
User can sign up a personal account.<br>
Post their favorite camping place.<br>
Comment on the campground posts. 
<br>
